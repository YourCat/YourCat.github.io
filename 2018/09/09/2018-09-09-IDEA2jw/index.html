安装 Mysql 及设置用户名和密码

打开命令行依次输入以下三条：

sudo apt-get install mysql-server
sudo apt install mysql-client
sudo apt install libmysqlclient-dev

这样直接安装的Mysql因为版本问题没有设置密码的步骤，可以采用如下方式自行设置
输入：

sudo vi /etc/mysql/debian.cnf

打开Mysql的配置文件，找到用户名：debian-sys-maint，密码：aWn**********98（随机生成）
Esc退出，输入“：q！”

mysql -u debian-sys-maint -p
Enter password:(输入刚才的密码)

成功进入到数据库中，接下来设置root用户和密码

mysql> update mysql.user set authentication_string=PASSWORD(“ihan″) where User=’root’;
mysql> update mysql.user set plugin=”mysql_native_password”;
mysql> flush privileges;
mysql> quit;

退出后重启Mysql

sudo /etc/init.d/mysql stop
sudo /etc/init.d/mysql start

尝试用设置好的root账户登陆～

安装 Redis

安装Redis服务端：

sudo apt-get install redis-server

检查Redis服务器系统进程

ps -aux|grep redis

检查Redis服务器状态

netstat -nlt|grep 6379

如果报错Command ‘netstat’ not found
先sudo apt install net-tools

安装 Tomcat 及在IDEA上进行配置

在官网上下载需要的tomcat版本

https://tomcat.apache.org/download-80.cgi

解压到新目录下：

sudo mkdir /opt/tomcat
sudo tar -xzvf apache-tomcat-9.0.4.tar.gz
sudo mv apache-tomcat-9.0.4 /opt/tomcat/

以管理员权限进到bin目录下，然后

sudo vi startup.sh

添加如下配置

sudo vi shutdown.sh

然后在shutdown.sh，开启tomcat：

./startup.sh

在浏览器中观察8080的小黄毛，关闭tomcat：

./shutdown.sh
